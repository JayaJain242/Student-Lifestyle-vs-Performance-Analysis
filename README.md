# Student-Lifestyle-vs-Performance-Analysis

# Project Overview
This project analyzes the relationship between student lifestyle factors such as study hours,sleep,social media usage,and academic performance.

# Dataset
This dataset was created by me to represent real-life student behavior and performance patterns. This dataset includes:
Study Hours
Sleep Hours
Social Media Usage
Attendance
Marks
Stress Level
Category (Topper/Average/Low)

# Analysis Performed
Data cleaning and exploration using Python (Pandas)
Visualization using bar graphs, pie chart, and scatter plots
Group-wise comparison using category

# Key Insights
Students who study more tend to score higher marks
Higher social media usage is associated with lower performance
Balanced lifestyle leads to average to good performance
Attendance positively impacts marks
Some exceptions exist, indicating multiple influencing factors

Pandas

Jupyter Notebook
